📘 Book Management Web
เว็บแอปพลิเคชันสำหรับจัดการหนังสือ โดยผู้ใช้สามารถเพิ่ม แก้ไข และลบหนังสือได้ และมีระบบแยกสิทธิ์ระหว่าง User และ Admin

Features
- สมัครสมาชิก / เข้าสู่ระบบ / ออกจากระบบ
- เพิ่ม แก้ไข ลบ หนังสือ (CRUD)
- แยกสิทธิ์การใช้งานตาม Role
- Admin เห็นหนังสือทั้งหมด และเห็นว่าใครเป็นผู้เพิ่ม

วิธีรันโปรเจค
- pip install flask flask-sqlalchemy flask-login
- python app.py

Role System
- ผู้ใช้ทั่วไป (user) จะเห็นเฉพาะหนังสือของตนเอง
- ผู้ดูแลระบบ (admin) จะเห็นหนังสือทั้งหมด
- Admin จะเห็นชื่อผู้เพิ่มหนังสือกำกับไว้
